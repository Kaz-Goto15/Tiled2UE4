linkPath.jsonを編集する際の注意
・"tiled"には使用したタイルセットのファイル名をそのまま入れてください。
・"ue4"には、コンテンツブラウザ内でアセットをCtrl+Cなどでコピーを行うと PaperTileSet'/Game/Assets/tilemap/tileset_field_TileSet.tileset_field_TileSet' のようなパスがコピーされます。
　このうち、''で囲まれた部分を入れてください。（上記の例では、 /Game/Assets/tilemap/tileset_field_TileSet.tileset_field_TileSet の部分）

Tiledでマップを制作する際の注意
・タイルセット(.tsx)は必ずプロジェクトファイル(.tmx)と同階層に置いてください。
　・階層が違った場合、パスが変わるのでパーサーが作動しません。どうしても同階層に置きたくない場合は、都度path.jsonにパスを書いてください。書くべきパスはエクスポートしたjsonファイルにあります。

変換後の注意
・Tiledのタイルセットで使用した画像ファイルとUE4のタイルセットで使用した画像ファイル(テクスチャ)は同じ画像になるようにしてください。(画像が同じであればパスが違っても問題ありません)
　・差分があった場合、意図しない地形や空白の地形が入る可能性があります。
jsonをUE4にD&Dしてからパーサで出力されたテキストをコピペするまでの間は、必ず１つ以上のタイルレイヤーを残すようにしてください。
　・〇〇のコピー のレイヤーを修正するのが面倒だからといってD&D直後に全レイヤーを削除しようとするとUEがクラッシュします。
